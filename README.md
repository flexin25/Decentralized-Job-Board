# 🧑‍💻 Decentralized Job Board (Soroban Smart Contract)

## 📌 Project Description
<img width="1849" height="874" alt="image" src="https://github.com/user-attachments/assets/37b1397e-ad9b-4239-ad27-d0936e920573" />

This project is a decentralized job board built on the Stellar blockchain using Soroban smart contracts. It allows employers to post job listings on-chain and enables anyone to access job data in a transparent, trustless manner.

The goal is to remove intermediaries and create an open job marketplace powered by blockchain.

---

## ⚙️ What It Does
- Allows employers to post job listings directly on-chain  
- Stores job data permanently and transparently  
- Enables users to fetch individual jobs or all job listings  
- Ensures trustless interaction without centralized control  

---

## ✨ Features
- 📝 Post Jobs (Title, Description, Employer)
- 🔍 View Single Job by ID
- 📋 Fetch All Job Listings
- 🌐 Fully On-Chain Storage
- 🔐 Decentralized & Trustless
- ⚡ Built using Soroban (Stellar Smart Contracts)
